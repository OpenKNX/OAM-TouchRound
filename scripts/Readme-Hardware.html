Die Software für dieses Release wurde auf folgender Hardware getestet und läuft damit "out-of-the-box":

  • AB-SmartHome TouchRound-Display RP2040 Vorserienmodell
  • AB-SmartHome TouchRound-Display RP3050 AB-SmartHome Shop: Touch-Display

Andere Hardware kann genutzt werden, jedoch muss das Projekt dann neu compiliert werden. Alle notwendigen Teile für ein Aufsetzen der Build-Umgebung inclusive aller notwendigen Projekte finden sich im OpenKNX-Projekt

Interessierte sollten auch die Beiträge im OpenKNX-Forum studieren.
